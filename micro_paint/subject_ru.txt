Напишите программу, которая будет читать «файл операции» и выводить результат в терминал.

Ваша программа должна принимать один аргумент, это будет путь к "операционному файлу".
Если вашей программе дано 0 или более чем 1 аргумент, напишите «Ошибка: аргумент», за которым следует \ n в STDOUT.

$> ./micro_paint | cat -e
Error: argument$
$> ./micro_paint do not fear mathematics | cat -e
Error: argument$
$>

Если при открытии и / или чтении «файла операции» возникает какая-либо проблема,
напишите «Ошибка: файл операции поврежден», за которым следует \ n в STDOUT.

"Операционный файл" будет содержать строки по одной операции на строку.
Если строка неверна, возникает ошибка.
Если произошла ошибка, ваша программа должна вернуть 1
Если ошибок не было, он должен вернуть 0
Последняя строка может быть как с символом \ n, так и без него.
Строки должны читаться по порядку, поэтому операции должны выполняться в том же порядке.
Между каждой переменной в строке должен быть хотя бы один пробел.

ШИРИНА ВЫСОТА BACKGROUND_CHAR
Эта строка всегда является первой строкой файла и определяет зону, в которой нужно рисовать. Ваша программа не должна отображать ничего за пределами зоны рисования. Он должен быть там только один раз.
- WIDTH: должно быть целым числом с 0 <WIDTH <= 300, горизонтальным количеством символов для использования в зоне рисования.
- ВЫСОТА: должно быть целым числом с 0 <ВЫСОТА <= 300, количество символов по вертикали для использования в зоне рисования.
- BACKGROUND_CHAR: любое пустое пространство будет заполнено BACKGROUND_CHAR

r X Y ШИРИНА ВЫСОТА СИМВОЛ
Эта операция будет рисовать пустой прямоугольник, где рисуется только граница прямоугольника.
- r: символ r
- X: любой поплавок, горизонтальное положение левого верхнего угла прямоугольника
- Y: любой поплавок, вертикальное положение левого верхнего угла прямоугольника
- WIDTH: положительное значение с плавающей запятой, но не 0, ширина прямоугольника (по горизонтали)
- ВЫСОТА: положительное значение с плавающей запятой, но не 0, высота прямоугольника (по вертикали)
- СИМВОЛ: символ используется для рисования прямоугольника

R X Y ШИРИНА ВЫСОТА СИМВОЛ
Эта операция нарисует заполненный прямоугольник
- R: символ R
- X: любой поплавок, горизонтальное положение левого верхнего угла прямоугольника
- Y: любой поплавок, вертикальное положение левого верхнего угла прямоугольника
- WIDTH: положительное значение с плавающей запятой, но не 0, ширина прямоугольника (по горизонтали)
- ВЫСОТА: положительное значение с плавающей запятой, но не 0, высота прямоугольника (по вертикали)
- СИМВОЛ: символ используется для рисования прямоугольника

Зона рисования разделена на прямоугольники, каждый из которых может содержать по одному символу (поскольку мы находимся в терминале ...), мы будем называть их пикселями.
Чтобы все было проще, мы будем использовать только верхний левый угол пикселя, чтобы знать, находится ли этот пиксель в прямоугольнике или нет.
Пиксель с верхним левым углом на расстоянии больше или равного 1 от границы прямоугольника не является частью пустого прямоугольника.
Пиксель, левый верхний угол которого находится на расстоянии менее 1 от границы прямоугольника, является частью пустого прямоугольника.

Вы должны смотреть на изображение, читая следующие несколько строк. Он представляет собой терминал. Мы установили зону рисования ШИРИНОЙ 5 и ВЫСОТОЙ 3.
